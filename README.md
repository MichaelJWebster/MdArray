# MdArray

Multi Dimensional arrays for javascript.

## Getting Started
Install the module with: `npm install MdArray`

```javascript
var MdArray = require('MdArrayjs');
var a = MdArrayjs.([0, 1, 2, 3], 2, 2); // Create 2 X 2 array.
```
